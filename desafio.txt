🎯 Objetivo

Neste desafio, vocês deixarão de ser apenas "estudantes" e atuarão como Engenheiros de Software construindo o núcleo (Core) de um sistema de gestão para a nossa ONG parceira. O objetivo é aplicar os pilares da Programação Orientada a Objetos: Encapsulamento, Herança e Composição.

🏗️ Especificações do sistema (o que você deve codificar)

Seu sistema deve ser escrito em Java e conter, no mínimo, a seguinte estrutura arquitetural:

1. A superclasse: `MembroONG` (a base)

* Atributos Privados: `nome` (String), `cpf` (String) e `diasAtuacao` (int).

* Encapsulamento: Crie os métodos Getters e Setters.

* Regra de Negócio (Setter): O método `setDiasAtuacao(int dias)` não pode aceitar valores negativos.

* Comportamento: Crie um método `exibirResumo()` que imprima o nome e os dias de atuação.



2. A herança: `Voluntario` e `Doador` (a especialização)

* Crie a subclasse `Voluntario` que herda (`extends`) de `MembroONG`.

   * Atributo específico: `setor` (String - ex: "Logística", "Aulas").

   * Sobrescrita: Use `@Override` para reescrever o método `exibirResumo()`, incluindo agora o setor do voluntário.

* Crie a subclasse `Doador` que herda (`extends`) de `MembroONG`.

   * Atributo específico: `valorDoadoMensal` (double).

   * Sobrescrita: Use `@Override` para reescrever o método `exibirResumo()`, exibindo o status de "Doador Ativo" e o valor.



3. A composição: `ProjetoSocial` ("o lego")

* Regra do TEM-UM: Um projeto de extensão não é um voluntário, ele tem um voluntário líder.

* Crie a classe `ProjetoSocial`.

* Atributos: `nomeDoProjeto` (String), `metaImpacto` (int - número de pessoas atendidas).

* Composição: Adicione um atributo `lider` do tipo `Voluntario`.

* Comportamento: Crie um método `iniciarProjeto()` que imprima uma mensagem dizendo que o projeto começou, quem é o líder responsável e qual o setor dele.



4. A classe de teste: `SistemaMain`

* Crie a classe que contém o método `public static void main(String[] args)`.

* Neste método, você deve:

   1. Instanciar um `Voluntario` e um `Doador`, preenchendo seus dados via Setters.

   2. Instanciar um `ProjetoSocial`.

   3. Atribuir o voluntário criado como líder desse projeto.

   4. Chamar os métodos `exibirResumo()` das pessoas e o `iniciarProjeto()` do projeto para provar que tudo funciona em harmonia.
